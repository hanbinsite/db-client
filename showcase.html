DB-CLIENT - 实现展示

项目概述

本项目实现了一个模仿Navicat风格的数据库客户端界面，具有现代化的UI设计和完整的主题切换功能。

核心功能

  • 完整的主题切换系统（亮色/暗色模式）
  • 数据面板 - 包含表格展示、列控制、过滤等功能
  • 连接面板 - 管理数据库连接
  • 数据库面板 - 浏览数据库对象
  • 查询面板 - 执行SQL查询
  • 专业的Navicat风格UI设计

项目结构

{`db-client/ ├── src/ │ ├── main/ # Electron主进程代码 │ └── renderer/ # React渲染进程代码 │ ├── components/ # UI组件 │ │ ├── ThemeContext.tsx # 主题上下文 │ │ ├── DataPanel.tsx # 数据面板 │ │ ├── ConnectionPanel.tsx # 连接面板 │ │ ├── DatabasePanel.tsx # 数据库面板 │ │ └── QueryPanel.tsx # 查询面板 │ ├── App.tsx # 主应用组件 │ └── index.tsx # 应用入口 └── showcase.html # 此展示页面`}

核心组件

ThemeContext.tsx

全局主题上下文，提供亮色/暗色模式切换和持久化存储

DataPanel.tsx

数据展示面板，支持表格操作、列控制、过滤、排序等功能

ConnectionPanel.tsx

数据库连接管理面板，支持连接的创建、编辑、删除

DatabasePanel.tsx

数据库对象浏览面板，展示数据库、表、视图等对象

QueryPanel.tsx

SQL查询编辑器，支持查询执行和结果展示

主题系统实现

主题系统基于React Context API实现，提供了以下特性：

  • 全局主题状态管理
  • 本地存储持久化
  • 自定义useTheme Hook便于组件使用
  • 自动应用主题类到body元素
{`// 使用useTheme Hook访问主题状态 const { darkMode, toggleDarkMode } = useTheme(); // 应用主题到组件
...
`}

数据面板功能

DataPanel组件实现了以下功能：

  • 表格数据展示
  • 列显示/隐藏控制
  • 数据过滤功能
  • 排序功能
  • 分页控制
  • 数据导出和复制
  • 记录编辑和新增

使用指南

由于网络环境限制，完整依赖安装可能会遇到问题。若要运行项目，请按照以下步骤操作：

  1. 确保Node.js已安装
  2. 在项目根目录执行 npm install
  3. 安装完成后，执行 npm run dev

如果依赖安装失败，可以直接查看代码实现，所有功能都已经在src目录下完整实现。
